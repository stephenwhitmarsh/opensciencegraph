# IBL - International Brain Laboratory

## Overview
The International Brain Laboratory is a global consortium of ~20 experimental and theoretical neuroscience laboratories that operates under an explicit policy of releasing all collected data and code publicly. Founded in 2017 with support from the Simons Foundation and Wellcome Trust, IBL was designed from the outset as an open science collaboration — data sharing is not an afterthought but a core structural commitment. Its flagship output, the Brain Wide Map, provides the first brain-wide atlas of neural activity during a standardised decision-making task, fully released on [[Platforms/DANDI Archive\|DANDI Archive]] in [[Standards/NWB\|NWB]] format.

## Open Science Commitments
- All experimental data released publicly on [[Platforms/DANDI Archive\|DANDI Archive]] in [[Standards/NWB\|NWB]] format
- All analysis code released on GitHub under open licences
- Standardised task, hardware, and surgical protocols shared openly to enable cross-lab reproducibility
- Pre-registration of analyses before data collection in several projects
- Brain Wide Map dataset: Neuropixels recordings from 547 brain regions across ~200 sessions, freely available

## Key Outputs
- IBL Brain Wide Map (2023) — brain-wide electrophysiology atlas, released on DANDI
- Standardised IBL task protocol (freely available)
- PyBpod and ONE (Open Neurophysiology Environment) — open data access API
- Reproducibility study: replicated a single experiment across 9 labs to quantify inter-lab variability
