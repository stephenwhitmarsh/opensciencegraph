# HED - Hierarchical Event Descriptors

## Overview
HED is a structured annotation system for experimental events in neuroscience data,
providing a hierarchical vocabulary to describe events in a machine-readable way.
Fully integrated into [[Standards/BIDS\|BIDS]] event files and endorsed by [[Actors/INCF\|INCF]].
The SCORE extension provides clinical EEG annotations.

## Created / Governed By
- [[Actors/INCF\|INCF]] — endorsing organisation
- [[Governance/INCF Working Groups\|INCF Working Groups]] — developed through INCF task forces

## Connections
- Endorsed by: [[Actors/INCF\|INCF]]
- Integrated with: [[Standards/BIDS\|BIDS]], [[Standards/NWB\|NWB]], [[Platforms/OpenNeuro\|OpenNeuro]]
- Complements: [[Standards/Cognitive Atlas\|Cognitive Atlas]] (cognitive task semantics),
  [[Standards/openMINDS\|openMINDS]] (EBRAINS metadata), [[Standards/UBERON\|UBERON]] (brain region annotation)
- Clinical extension: SCORE (seizure and clinical EEG annotation)
