# RDA - Research Data Alliance

## Standards Produced
- [[Governance/FAIR Principles\|FAIR Principles]] — FAIR Data Maturity Model Working Group output

## Overview
RDA builds social and technical infrastructure for open data sharing through community Working Groups and Interest Groups producing concrete outputs in 18-month cycles. The [[Governance/RDA Neuroscience IG\|RDA Neuroscience IG]] connects RDA to brain data standards. French national node via [[Governance/CNRS Open Science\|CNRS Open Science]].

## Relevant Groups for Neuroscience
- [[Governance/RDA Neuroscience IG\|RDA Neuroscience IG]]
- FAIR Data Maturity Model WG
- Health Data IG
- Persistent Identifiers IG
