# GO FAIR

## Overview
GO FAIR is a bottom-up stakeholder-driven initiative providing the Three-Point FAIRification Framework for implementing [[Governance/FAIR Principles\|FAIR Principles]]. It operates through Implementation Networks (INs) including a Neuroscience IN. A French GO FAIR office operates via [[Governance/CNRS Open Science\|CNRS Open Science]].

## Key Outputs
- Three-Point FAIRification Framework (Find, Interoperate, Reuse)
- GO FAIR Implementation Networks (INs) across multiple disciplines
- GO FAIR Neuroscience IN — coordinates FAIR implementation in neuroscience
