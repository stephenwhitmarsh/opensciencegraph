# OMERO — OME Remote Objects

## Overview
OMERO is the client-server image data management platform produced by the [[Actors/OME\|OME]] consortium. It is to microscopy image data what [[Platforms/LORIS\|LORIS]] and [[Platforms/XNAT\|XNAT]] are to neuroimaging — the institutional platform where raw images are ingested, annotated, quality-checked, and made accessible to researchers. OMERO is deployed at [[Governance/Euro-BioImaging\|Euro-BioImaging]] nodes, [[Governance/France BioImaging\|France BioImaging]] nodes (including [[Actors/Institut Pasteur\|Institut Pasteur]] and [[Actors/Institut Curie\|Institut Curie]] PICT-IBiSA), and core facilities worldwide.

## Key Features
- Server-based management of terabyte-scale image collections
- Role-based access control for collaborative and public data sharing
- In-browser image viewer (OMERO.web) supporting multi-dimensional data
- Python, Java, MATLAB, and R APIs for programmatic access
- Bio-Formats integration: imports 150+ proprietary microscopy formats
- Metadata annotation via key-value pairs, tags, and structured annotations
- OMERO.figure — publication-quality figure creation from managed images
- Links to BioImage Archive for public deposition of managed datasets
