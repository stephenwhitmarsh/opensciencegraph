# BrainLife.io

## Overview
BrainLife is an open cloud computing platform for FAIR and reproducible neuroimaging analysis. Users run containerised Apps on [[Standards/BIDS\|BIDS]]-organised data with automatic provenance tracking. It directly imports from [[Platforms/OpenNeuro\|OpenNeuro]] and [[Platforms/DANDI Archive\|DANDI Archive]].

## Connections
- Standards: [[Standards/BIDS\|BIDS]], [[Standards/NWB\|NWB]]
- Integrates with: [[Platforms/OpenNeuro\|OpenNeuro]], [[Platforms/DANDI Archive\|DANDI Archive]]
- FAIR compliance: via [[Standards/BIDS\|BIDS]] format and [[Actors/ReproNim\|ReproNim]] provenance tracking
