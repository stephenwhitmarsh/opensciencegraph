# DANDI Archive

## Overview
DANDI is the primary public archive for neurophysiology data (NIH BRAIN Initiative). It requires [[Standards/NWB\|NWB]] format and provides FAIR-compliant access with persistent DOIs, cloud storage (AWS), versioned datasets, and DANDI CLI for programmatic access.

## Connections
- Primary format: [[Standards/NWB\|NWB]]
- Related platforms: [[Platforms/OpenNeuro\|OpenNeuro]], [[Platforms/BrainLife.io\|BrainLife.io]]
- Funded by: [[Actors/NIH BRAIN Initiative\|NIH BRAIN Initiative]]
- Key depositors: [[Actors/IBL\|IBL]] (Brain Wide Map), [[Actors/Allen Institute for Brain Science\|Allen Institute for Brain Science]]
- FAIR access: persistent DOIs, versioned datasets, CC0 licence via [[Standards/NWB\|NWB]] format
