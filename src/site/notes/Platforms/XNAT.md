# XNAT - Extensible Neuroimaging Archive Toolkit

## Overview
XNAT is an open-source imaging informatics platform for DICOM management, project organisation, pipeline execution via Container Service (Docker), and data sharing. Used at multiple [[Platforms/EBRAINS\|EBRAINS]] member sites.
