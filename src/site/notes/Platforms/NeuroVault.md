# NeuroVault

## Overview
NeuroVault is a public repository for sharing unthresholded statistical maps, parcellations, and atlases from neuroimaging studies. It enables replication and meta-analysis via Neurosynth integration, complementing [[Platforms/OpenNeuro\|OpenNeuro]] (raw data).

## Connections
- Standards: [[Standards/BIDS\|BIDS]], [[Standards/NIDM\|NIDM]]
- Integrates with: [[Platforms/OpenNeuro\|OpenNeuro]], Neurosynth, [[Standards/Cognitive Atlas\|Cognitive Atlas]]
- FAIR access: persistent DOIs, open licences, DOI-linked statistical maps
