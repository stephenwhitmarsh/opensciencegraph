# RDMkit — ELIXIR Research Data Management Toolkit

## Overview
RDMkit is [[04_Governance/ELIXIR\|ELIXIR]]'s open online resource providing guidance, best practices, and tool recommendations for research data management (RDM) in the life sciences. Launched in 2021 and maintained by the ELIXIR community with contributions from all national nodes (including [[04_Governance/IFB\|IFB]] for the French context), RDMkit is organised
around the data lifecycle and covers every stage from planning and collecting through to preserving and reusing research data.

RDMkit serves as the practical implementation guide for [[04_Governance/FAIR Principles\|FAIR Principles]] in life sciences research — translating abstract FAIR requirements into concrete tool recommendations, workflow descriptions, and domain-specific guidance.

## Structure

### By Role
RDMkit provides tailored guidance for different roles in a research project:
- **Principal Investigators** — DMP writing, data governance, funder compliance
- **Data Stewards** — FAIR implementation, metadata, repository selection
- **Researchers** — day-to-day data organisation, file naming, documentation
- **IT support** — infrastructure, storage, security

### By Domain
Domain-specific pages provide guidance for particular life science communities:
- **Bioimaging** — microscopy data, [[02_Standards/OME File Formats\|OME File Formats]], [[02_Standards/REMBI\|REMBI]] metadata
- **Genomics** — sequencing data, [[02_Standards/VCF\|VCF]], [[03_Platforms/EGA\|EGA]]/[[03_Platforms/NCBI GEO\|NCBI GEO]] deposition
- **Proteomics, metabolomics** — omics data standards
- **Human data** — GDPR compliance, sensitive data, [[03_Platforms/EGA\|EGA]] for restricted data
- **Rare diseases** — [[02_Standards/ORDO\|ORDO]], [[02_Standards/HPO\|HPO]], [[02_Standards/ERN Vocabularies\|ERN Vocabularies]], patient registry data

### By Country
National pages provide country-specific guidance, tools, and contacts:
- **France** — [[04_Governance/IFB\|IFB]] tools (FAIR-Checker, DMP OPIDoR), [[04_Governance/OPIDoR\|OPIDoR]], [[04_Governance/HAL\|HAL]],
  [[03_Platforms/Recherche Data Gouv\|Recherche Data Gouv]], [[04_Governance/Health Data Hub\|Health Data Hub]] for health data

### Tool Assembly
RDMkit's **Tool Assembly** pages describe complete RDM workflows for specific contexts — e.g. the IFB Tool Assembly describes the complete French life sciences RDM stack from data production to archive.
