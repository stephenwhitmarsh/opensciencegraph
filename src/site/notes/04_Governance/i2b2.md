# i2b2 - Informatics for Integrating Biology and the Bedside

## Overview
i2b2 is an open-source clinical data warehousing and query framework widely deployed at hospitals globally. [[04_Governance/AP-HP\|AP-HP]] runs an i2b2-based CDW alongside its [[02_Standards/OMOP CDM\|OMOP CDM]] EDS warehouse. Integrates with [[02_Standards/HL7 FHIR\|HL7 FHIR]].

## Connections
- Compatible with: [[02_Standards/OMOP CDM\|OMOP CDM]] (migration pathway), [[02_Standards/HL7 FHIR\|HL7 FHIR]]
- Deployed at: [[04_Governance/AP-HP\|AP-HP]], [[04_Governance/Health Data Hub\|Health Data Hub]] (indirect)
