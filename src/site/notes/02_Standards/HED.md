# HED - Hierarchical Event Descriptors

## Overview
HED is a structured annotation system for experimental events in neuroscience data, providing a hierarchical vocabulary to describe events in a machine-readable way. Fully integrated into [[02_Standards/BIDS\|BIDS]] event files and endorsed by [[01_Actors/INCF\|INCF]], the SCORE extension provides clinical EEG annotations.

## Created / Governed By
- [[01_Actors/INCF\|INCF]] — endorsing organisation
- [[04_Governance/INCF Working Groups\|INCF Working Groups]] — developed through INCF task forces

## Connections
- Endorsed by: [[01_Actors/INCF\|INCF]]
- Integrated with: [[02_Standards/BIDS\|BIDS]], [[02_Standards/NWB\|NWB]], [[03_Platforms/OpenNeuro\|OpenNeuro]]
- Complements: [[02_Standards/Cognitive Atlas\|Cognitive Atlas]] (cognitive task semantics), [[02_Standards/openMINDS\|openMINDS]] (EBRAINS metadata), [[02_Standards/UBERON\|UBERON]] (brain region annotation)
- Clinical extension: SCORE (seizure and clinical EEG annotation)
