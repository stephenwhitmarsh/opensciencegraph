# DANDI Archive

## Overview
DANDI is the primary public archive for neurophysiology data (NIH BRAIN Initiative). It requires [[02_Standards/NWB\|NWB]] format and provides FAIR-compliant access with persistent DOIs, cloud storage (AWS), versioned datasets, and DANDI CLI for programmatic access.

## Connections
- Primary format: [[02_Standards/NWB\|NWB]]
- Related platforms: [[03_Platforms/OpenNeuro\|OpenNeuro]], [[03_Platforms/BrainLife.io\|BrainLife.io]]
- Funded by: [[01_Actors/NIH BRAIN Initiative\|NIH BRAIN Initiative]]
- Key depositors: [[01_Actors/IBL\|IBL]] (Brain Wide Map), [[01_Actors/Allen Institute for Brain Science\|Allen Institute for Brain Science]]
- FAIR access: persistent DOIs, versioned datasets, CC0 licence via [[02_Standards/NWB\|NWB]] format
